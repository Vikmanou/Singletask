[![License](https://img.shields.io/github/license/vikmanou/singletask?style=flat-square)](LICENSE)

A minimal, distraction-free Pomodoro timer — available as a desktop app and in the browser.

---

## Features

- **25-minute Pomodoro timer** with start, pause, and resume
- **Adjust time** in 1-minute increments
- **Alarm sound + desktop notification** when the session ends
- **5 themes** — Void, Deep Ocean, Campfire, Moonlit Desk, Greenhouse
- **Always on top** toggle (Electron only) — pin the window above all other apps

---

## Keyboard Shortcuts

| Key | Action |
|---|---|
| `Space` | Start / Pause / Resume |
| `↑` | Add 1 minute |
| `↓` | Remove 1 minute |
